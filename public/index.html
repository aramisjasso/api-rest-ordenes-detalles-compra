API de Tareas

Esta es una interfaz para interactuar con la API de tareas.

Lista de Tareas

    Crear Nueva Tarea

    Documentación de la API

    Obtener todas las tareas

    GET /apiV1/task

    Devuelve una lista de todas las tareas.

    Obtener una tarea por ID

    GET /apiV1/task/{id}

    Devuelve una tarea específica por su ID.

    Crear una nueva tarea

    POST /apiV1/task

    Crea una nueva tarea. El cuerpo de la solicitud debe ser un JSON con el campo title.

    {
    "title": "Nueva tarea"
}

    Actualizar una tarea

    PUT /apiV1/task/{id}

    Actualiza una tarea existente. El cuerpo de la solicitud debe ser un JSON con el campo title.

    {
    "title": "Tarea actualizada"
}
